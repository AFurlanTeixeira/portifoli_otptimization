# portifoli_otptimization
Aqui se encontram os dados e também, futuramente, o relatório de um Experimento Experimental Estatístico utilizando o Método Taguchi aplicado à otimização de portifólios de investimento (especificamente voltado para ações da Bolsa de Valores Brasileira, B3). 

Como base de dados, vou utilizar o Yahoo Finance, através da biblioteca yfinance no Python para trazer séries históricas reais sobre os ativos que farão parte do portifólio
